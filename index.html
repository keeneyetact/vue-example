In this example we introduce a whole bunch of new concepts: handling user input with v-on:click on the button, conditional rendering for paragrapsh via v-if, conditional display using v-show, and finally loops in templates via v-for. Look at the source code of this page for more details. Note the difference between v-if, which provide a real conditional rendering, and v-for, which simply hides and shows the element depending on the expression's truthness.
